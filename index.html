Hexo + Github Pages

博客架设于github；静态博客hexo；markdown写作。数据归自己管，可以本地随时查看，对外是博客，对内知识库。

1. Github配置

  • 注册Github账号
    访问地址：https://github.com
    用户名：[uname]
    邮箱：[user]@yic.ac.cn
    密码：[passwd]
    成功后，访问地址：https://[uname].github.com

  • 创建代码库
    New repository …
    作为博客使用,则repository name名字填写：uname.github.io

  • 开启GitHub Pages功能
    Settings -> GitHub Pages …

  • 完成后，可以正常访问：http://[uname].github.io
