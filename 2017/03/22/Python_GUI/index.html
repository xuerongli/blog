图形用户界面（Graphical User Interface，简称 GUI）是指采用图形方式显示的计算机操作用户接口。

Python的GUI工具包：

  • Tkinter
  • wxPython

  • PyQt

  • 创建主窗体
  • 创建元件
  • 显示元件
